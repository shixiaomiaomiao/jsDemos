         {
           var aResult = [];

  • 课时1   Ajax基础

      • 课堂笔记:


      1.Ajax:无刷新读取服务器上的数据
      2.Ajax读取静态文件调用ajax.js(文件放于服务器的www中),function ajax(url,fnSucc,fnFailed)
       nbsp url:要读取的文件的路径,fnSucc:读取成功以后执行的函数,fnFailed:读取文件失败以后执行的文件。
      eg:读取文件并将文件内容返回:ajax("aa.txt",function(str){alert(str)},function(){alert("失败")});
      注：在执行该html文件时，路径为：localhost/...(绝对路径将无法读取),另外存的文件名必须是英文的
      3.字符集编码:要求html,js,和要读取的文件均是同一个编码的。建议使用utf-8编码。(另外还有中文编码:GB2312)
      4.缓存:对于同一个地址只读取一次.
        阻止缓存:一种方法是让URL地址一直变化.
      eg:ajax("aa.txt ? = " + new Date().getTime(),function(str){alert(str)},function(){alert("失败")}).
      5.用ajax读取过来的任何数据都是以字符串的形式存在
      6.eval(str):可以将str解析成JS可以识别的东西。
      7.表单提交的两种方法(method):get 和 post
        get:通过网址上传(将数据放入url里，基本格式是：名字=值&名字=值...)，容量小(不适合上传大数据)，安全性差，有缓存。
        post:通过http Content传递,容量大，安全性好一点，没有缓存。
      一般,get方式用于获取数据(如浏览帖子)。post方式用于上传数据(如用户注册)

  • 课时2   Ajax中级

      • 课堂笔记：


      1.Ajax的使用步骤:（类似于打电话）
      (1)创建Ajax对象（需要有手机这个物体）
        var oAjax = new XMLHttpRequest();//在非IE6下兼容
        var oAjax = new ActiveXObject('Microsoft.XMLHTTP');//在非IE6下兼容
      (2)连接到服务器（拨号）
      (3)发送请求（告诉服务器）（说话）
      (4)接收返回值（听对方说话）
      具体函数代码:
      function ajax(url,fnSucc,fnFailed)
       {
          //创建一个对象
          if(window.XMLHttpRequest)
           {
             var oAjax = XMLHttpRequest();
           }
          else
           {
            var oAjax = new ActiveXObject('Microsoft.XMLHTTP');
          }
             //连接服务器（拨号）open(方法,文件名,异步传输<一般都是异步>)
             //异步传输:多个事情可以一起做;同步传输：事件一件件的做
             oAjax.open('GET',url,true);
            
             //发送请求（告诉服务器）（说话）
             oAjax.send();
             //接收返回值（听对方说话）
             oAjax.onreadystatechange= function()
               {
                 //返回的5种值:0:(未初始化)还没有调用open()方法
                 //1：（载入）已调用send()方法，正在发送请求
                 //2：（载入完成）send()方法完成,已收到全部响应内容
                 //3:(解析)正在解析响应的内容
                 //4:（完成）响应内容解析完成（不代表成功），可以在客户端调用
                 if(oAjax.readyState == 4)
                   {
                     fnSucc(oAjax.responseText);
                   }
                   else
                   {
                     if(fnFailed)
                   {
                     fnFailed(oAjax.status);
                   }
                   }
                 }
          }

      字符串的操作

      1.查找并返回位置:search,用法: str.search(需要查找的字符).如果需要查找的是一个字符，找到则返回该字符在字符串中的位置，找不到则返回-1.如果需要查找的是一个字符串，若在str中找到完全相同的字符串，则返回该字符串的的首字符在str中的位置。若找不多，则返回-1.
      2.截取:subString,用法:str.subString(起始位置，终止位置)。若起始位置的数小于0，则从0开始；若终止位置的数大于str.length则按str.length来算;若终止位置的数不写的话，默认为str.length;
      3.查找并返回字符:charAt,用法:str.charAt(字符所在的位置)，若位置超出字符串，则返回空
      4.分割字符串:split,用法:str.split(分割的标志)，将字符串切成数组。
